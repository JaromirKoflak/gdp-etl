2 GDP Data Processing Script Documentation

This document provides comprehensive documentation for the GDP data processing R script.

2.1 Overview

This R script performs ETL (Extract, Transform, Load) operations on GDP data sourced from the UN Statistics Division. It handles preprocessing, correction of inconsistencies, data aggregation for regions/groups, and visualization of GDP trends over time.

2.2 1. Setup

2.2.1 Libraries

The script uses the following R packages: - tidyverse - readxl - httr - gridExtra - plotly

2.2.2 Working Directory

The script automatically sets the working directory to the location of the current R script file.

2.2.3 Directories

Defines paths for: - datadir: Data input - outputdir: Output files

2.3 2. Data Inputs

2.3.1 Country Metadata

  • Dim_countries.csv: Contains country codes and valid year ranges.
  • Dim_Countries_Hierarchy_All.csv: Contains hierarchical grouping for aggregations.

2.4 3. Functions

2.4.1 get_unsd_gdp_data()

  • Downloads GDP data from UN API (both constant and current prices).
  • Merges, reshapes, and formats the dataset.

2.4.2 compute_missing_values(df)

  • Handles historical and geopolitical inconsistencies by merging country records (e.g., Yugoslavia, USSR).
  • Aggregates GDP for reconstructed economies.

2.4.3 delete_data_out_of_valid_range(df)

  • Filters out data points falling outside the valid year range for each country.

2.4.4 expand_hierarchy(df)

  • Recursively expands hierarchical groupings (e.g., Americas, Europe).
  • Ensures all members of each group are captured for aggregation.

2.4.5 compute_aggregate_values(df)

  • Computes GDP aggregates for groups using hierarchical mappings.

2.4.6 add_economy_labels(df)

  • Joins human-readable economy labels.

2.4.7 add_notes(df)

  • Adds a placeholder “Note” column.

2.4.8 export_to_csv(df, filename)

  • Saves the final dataset to a CSV file.

2.5 4. Data Processing Pipeline 

input_data = get_unsd_gdp_data()
dfgdp = input_data %>%
  compute_missing_values() %>%
  delete_data_out_of_valid_range() %>%
  add_economy_labels() %>%
  compute_aggregate_values() %>%
  add_notes() %>%
  export_to_csv("gdp_update.csv")
